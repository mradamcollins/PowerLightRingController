light_ring

Arduino Sketch for a custom LED light_ring project

d15 Red
d14 Blue

d13
d12
d11
d10
d9
d8

Outter ring
d7

d5
d4
d3
d2

Note:
* LEDS are ganged in pairs on opposite sides
* d2 and d8 are roughly 90 degrees apart
